# 通用Web应用

一个使用原生HTML、CSS和JavaScript构建的现代化响应式网站项目。

## 项目特点

- 🚀 **纯原生技术** - 无需任何框架，使用HTML5、CSS3和现代JavaScript
- 📱 **响应式设计** - 完美适配桌面、平板和手机设备
- 🎨 **现代UI** - 采用渐变色彩和流畅动画
- ⚡ **高性能** - 优化的代码和资源加载
- 🔧 **模块化** - 清晰的文件结构和代码组织

## 项目结构

```
demo/
├── index.html          # 主页面
├── css/
│   └── styles.css      # 样式文件
├── js/
│   └── main.js         # JavaScript功能
└── README.md           # 项目说明
```

## 功能特性

### 🎯 核心功能
- **响应式导航栏** - 桌面端水平布局，移动端汉堡菜单
- **平滑滚动** - 页面内锚点导航和返回顶部
- **作品集筛选** - 动态筛选不同类型的项目展示
- **联系表单** - 带验证的表单提交功能
- **通知系统** - 优雅的消息提示组件

### 🎨 设计特色
- **渐变主题** - 紫蓝渐变的视觉风格
- **卡片布局** - 现代化的卡片式设计
- **悬停效果** - 丰富的交互动画
- **加载动画** - 页面元素渐入效果

### 📱 响应式特性
- **移动优先** - 从小屏幕开始设计
- **断点适配** - 768px和480px断点优化
- **触摸友好** - 适合移动设备的交互设计

## 技术栈

- **HTML5** - 语义化标签和现代API
- **CSS3** - Flexbox、Grid、动画和过渡
- **JavaScript ES6+** - 现代语法和DOM操作
- **Google Fonts** - Inter字体家族

## 快速开始

1. **克隆或下载项目**
   ```bash
   # 如果是git仓库
   git clone [repository-url]
   cd demo
   ```

2. **直接在浏览器中打开**
   ```bash
   # 使用任意HTTP服务器
   npx serve .
   # 或者
   python -m http.server 8000
   ```

3. **访问应用**
   打开浏览器访问 `http://localhost:8000`

## 自定义配置

### 🎨 主题颜色

```css

    --primary-color: #667eea;
```
