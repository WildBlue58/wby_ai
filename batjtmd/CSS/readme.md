## Shrink 收缩计算

- 溢出空间 300px 让子元素来shrink 分摊
- 分摊的比例？
  flex-shrink: 2:1 500:400

  500 - 300 X 500 需要分摊的比例
  400 - 300 X 400 需要分摊的比例
- 分摊的权重
  总权重？
  500 X 2 + 400 X 1 = 1400
  1000/1400 = 0.714 71.4%
  400/1400 = 0.286 28.6%
  
  300 X 0.714 = 214.28px
  300 X 0.286 = 85.72px

  left 500 - 214.28 = 285.72
  right 500 - 85.72 = 314.28

  ## Grow

  100 X 2/3 = 66.7
