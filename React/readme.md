## **项目**的创建

- npm是什么？ Node package manager
  node 包管理器 安装React package App开发能力
- npm init vite
  按 vite 模板初始化项目 init
  vite vue/react 项目模板和**工程**化
- 选择一些配置
  - react
  - JavaScript
  项目模板，基于它开始开发
- cd 项目名
- npm install 安装依赖
  node_modules 包所在
- npm run dev
  3000 端口 启动应用
