最长公共子序列 (LCS)

动态规划算法演示

🎯 核心思想

最值问题用动态规划 - 通过将复杂问题分解为子问题，利用最优子结构性质求解

🔧 DP特点

递归 → 迭代

先以递归自顶向下发现子问题之间的树状结构，找到状态转移方程

自底向上

DP自底向上去迭代问题，避免重复计算

最优子结构

局部最优与全局最优一致，dp[i][j] 表示状态

📋 动态规划五步曲

确定dp数组的含义

dp[i][j] 表示text1[0...i-1]和text2[0...j-1]的最长公共子序列长度

确定递推公式

if (text1[i-1] == text2[j-1]) dp[i][j] = dp[i-1][j-1] + 1
else dp[i][j] = max(dp[i-1][j], dp[i][j-1])

确定初始化

dp[0][j] = 0, dp[i][0] = 0
空字符串与任何字符串的LCS为0

确定遍历顺序

从左到右，从上到下
保证dp[i-1][j-1]等前置状态已计算

确定结果

返回dp[m][n]
m和n分别为两个字符串的长度

💡 算法复杂度

时间复杂度: O(m × n) // m, n为两个字符串长度
空间复杂度: O(m × n) // dp数组空间
优化后空间复杂度: O(min(m, n)) // 滚动数组优化
