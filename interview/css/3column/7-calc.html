CSS Calc() 函数布局

CSS calc() 函数允许在CSS中进行数学计算，特别适合需要精确控制尺寸的布局场景。在三栏布局中，calc()可以精确计算主内容区域的宽度，避免浮动元素重叠的问题。

核心特点

  • 支持加减乘除四则运算
  • 可以混合使用不同的单位
  • 实时计算，响应式友好
  • 现代浏览器支持良好

关键技术点

主内容区域宽度：width: calc(100% - 200px - 250px - 20px);

左侧边栏：width: 200px; float: left;

右侧边栏：width: 250px; float: right;

间距处理：margin: 0 10px;为左右各留10px间距

Calc() 函数示例

基础计算
width: calc(100% - 200px);
总宽度减去200像素
混合单位
width: calc(50vw - 2rem);
视口宽度的一半减去2个rem
复杂计算
width: calc((100% - 20px) / 3);
三等分宽度，减去间距
嵌套计算
width: calc(100% - calc(200px + 2rem));
嵌套calc函数的使用

响应式Calc()应用

在不同断点下，calc()函数可以动态调整布局：

桌面端 (>768px)
width: calc(100% - 450px);
平板端 (768px)
width: calc(100% - 200px);
移动端 (<768px)
width: 100%;
