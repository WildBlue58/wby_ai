CSS Position 属性详解

1. Static 定位（默认值）

Static
.static-box { position: static; /* 默认值，可以省略 */ }
特点：元素按照正常文档流排列，不受top、right、bottom、left属性影响

2. Relative 定位

原始位置
Relative
.relative-box { position: relative; top: 20px; /* 向下偏移20px */ left: 20px; /* 向右偏移20px */ }
特点：相对于元素原本位置进行偏移，不脱离文档流，原本位置仍然保留

3. Absolute 定位

容器内
Absolute
.absolute-container { position: relative; /* 为absolute子元素提供定位上下文 */ } .absolute-box { position: absolute; top: 50px; /* 距离容器顶部50px */ right: 50px; /* 距离容器右侧50px */ }
特点：相对于最近的定位祖先元素定位，完全脱离文档流，不占用空间

4. Fixed 定位

Fixed
.fixed-box { position: fixed; top: 20px; /* 距离视口顶部20px */ right: 20px; /* 距离视口右侧20px */ z-index: 1000; /* 确保在其他元素之上 */ }
特点：相对于浏览器视口定位，即使页面滚动位置也不变，完全脱离文档流

特点：在relative和fixed之间切换，需要指定top、right、bottom、left中的至少一个值

6. 其他值

/* inherit - 继承父元素的position值 */ .inherit-box { position: inherit; } /* initial - 设置为默认值static */ .initial-box { position: initial; } /* unset - 如果是继承属性则继承，否则设置为初始值 */ .unset-box { position: unset; }

7. 使用场景

  • static：默认布局，正常文档流
  • relative：微调元素位置，不影响其他元素
  • absolute：弹出框、工具提示、绝对定位的UI组件
  • fixed：固定导航栏、返回顶部按钮、模态框
  • sticky：粘性导航、表格头部固定
